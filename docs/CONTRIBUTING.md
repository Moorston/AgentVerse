# Contributing

We welcome contributions from:
- AI Researchers
- Agent Framework Developers
- GraphRAG Engineers
- Knowledge Graph Experts
- Open Source Contributors

## Getting Started

1. Read `DEVELOPMENT.md` for setup instructions
2. Check the project's Issues and Roadmap
3. Fork the repository and create a feature branch
4. Submit a Pull Request

## Code Style

- Python: Ruff (lint + format), Mypy strict mode
- TypeScript: ESLint + Prettier (via Next.js)
- Pre-commit hooks are configured — run `pre-commit install`
